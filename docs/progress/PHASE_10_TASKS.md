## 🛠️ 1. 導入・設定機能 (Config Setup)

### [10-1] 文言の変数化（ラベル定義の外出し）
- **優先度**: 🔴 高
- **ステータス**: 未着手
- **概要**: `src/config/labels.ts` を作成し、主要なUI文言（"Patient", "Chart" 等）を定数として管理します。
- **目的**: 汎用的な「Clinic CRM Lite」として、異なる業種（サロン等）でも用語を変更しやすくするため。
- **Note**: 全ての文言を置換するのではなく、主要な名詞から段階的に適用します。

### [10-2] バックアップ機能 (Export)
- **優先度**: 🟡 中
- **ステータス**: 未着手
- **概要**: 全データを CSV または SQL 形式でエクスポートする管理者向け機能を作成します。
- **目的**: データのポータビリティを確保し、ユーザーが安心して利用できるようにするため。
